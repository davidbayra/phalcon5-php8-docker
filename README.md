## Description
Simple starter Web-project for PHP Phalcon:
- 🦅 Phalcon framework version 5.1.4
- 🐘 PHP 8.1
- 🗄 MySql 8

### Configuration
⚠️ You need at least 2GB of memory for Docker.

to build:
```shell
docker-compose build
```

to start:
```shell
docker-compose up
```

and go to web 🕸: 
```
http://localhost:80
```
